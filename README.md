# Basic English Conversation Practice

![Status](https://img.shields.io/badge/status-live-886800) ![Gemini](https://img.shields.io/badge/AI-Gemini%202.0%20Flash-4285F4) ![Bilingual](https://img.shields.io/badge/language-EN%2FVI-f4e8a0)

> Agent #27 of 100 | 100 AI Agents Challenge

A friendly bilingual AI conversation partner for Vietnamese adults learning beginner English. Practice real scenarios — restaurants, neighbors, phone calls — with translations, corrections, and voice support.

## Features
- 6 real-life scenarios + free chat mode
- Every AI message shown in English + Vietnamese translation
- Gentle grammar corrections with Vietnamese explanation
- "I don't know what to say" helper with 3 suggested responses
- Type in Vietnamese → auto-translated to English
- Voice input + text-to-speech for all messages
